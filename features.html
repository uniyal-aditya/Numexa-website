Capabilities

Everything Numexa Can Do

A complete scientific computation suite — in your browser and in Discord.

∫
Calculus

Symbolic differentiation powered by math.js. Chain, product, quotient rules — all handled automatically.

derivative(x^4, x)           // 4 * x^3
derivative(sin(x)*x, x)      // cos(x)*x + sin(x)
derivative(log(x^2), x)      // 2 / x
⊞
Linear Algebra

Full matrix operations using array syntax. Supports any size matrix.

inv([[1,2],[3,4]])            // Inverse
det([[1,2],[3,4]])            // -2
transpose([[1,2],[3,4]])      // [[1,3],[2,4]]
lusolve([[2,0],[0,3]], [4,9]) // [2, 3]
📈
Function Plotter

Interactive canvas plotter with glowing curves, grid, and axis labels.

sin(x) + 0.5 * cos(2*x)
x^3 - 3*x
1 / (1 + exp(-x))   // sigmoid
🔢
Complex Numbers

Full complex arithmetic using i as the imaginary unit.

(3 + 4i) * (1 - 2i)    // 11 - 2i
abs(3 + 4i)            // 5
sqrt(-1)               // i
exp(pi * i)            // -1  (Euler)
📊
Statistics

Descriptive statistics on arrays — works with any list of numbers.

mean([2, 4, 6, 8, 10])       // 6
std([2, 4, 6, 8, 10])        // 2.83...
variance([1, 2, 3])          // 0.667
median([5, 1, 9, 3])         // 4
📐
Units & Constants

Built-in unit conversions and physical/mathematical constants.

5 km to m                    // 5000 m
100 mph to km/h              // 160.934...
pi                           // 3.14159...
phi                          // 1.61803... (golden ratio)
🤖
Discord Integration

All features available via Discord slash commands with rich embed responses.

/calc derivative(x^3, x)
/plot sin(x) * exp(-x/5)
/derive x^4 * log(x)  x
/matrix inv [[1,2],[3,4]]
⌨️
Combinatorics

Factorials, combinations, permutations and number theory.

factorial(10)                // 3628800
combinations(10, 3)          // 120
permutations(5, 2)           // 20
gcd(48, 18)                  // 6
