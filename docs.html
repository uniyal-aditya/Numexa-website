Getting Started

Numexa is a web-based scientific calculator and Discord bot powered by math.js. Open the Playground to use the calculator UI or write multi-line expressions in Type Mode.

Quick Start

// Basic arithmetic
2^10 + sqrt(144)

// Trigonometry (radians by default)
sin(pi / 6)   // → 0.5

// Derivatives
derivative(x^3 + 2*x, x)  // → 3x² + 2

Calculator

The Playground has two modes: Calculator UI (button grid) and Type Mode (multi-line editor).

Calculus

Symbolic derivatives are supported via the math.js derivative() function.

Derivatives

derivative(x^4, x)           // → 4x³
derivative(sin(x), x)        // → cos(x)
derivative(x*e^x, x)         // → e^x + x*e^x
derivative(log(x), x)        // → 1/x

Integration

Symbolic integration requires a server-side CAS engine (planned). Numerical integration coming in v1.3.

Linear Algebra

Use array syntax [[row1], [row2]] for matrices.

inv([[1,2],[3,4]])            // Inverse
det([[1,2],[3,4]])            // Determinant → -2
transpose([[1,2,3],[4,5,6]])  // Transpose
lusolve([[1,0],[0,1]], [1,2]) // Solve Ax=b

Statistics

Statistical functions accept arrays in [...] syntax.

mean([1,2,3,4,5])      // 3
std([1,2,3,4,5])       // 1.414...
variance([1,2,3])      // 0.667
median([3,1,4,1,5])    // 3

Discord Commands

Command Usage Description

/matrix /matrix inv [[1,2],[3,4]] Matrix operation
/help /help Show all commands
